Quick install

Install Nift v2.

Native binary. Single file. No runtime, no Node, no Python. Pick the path that matches your OS, then verify with nift --version. 

# macOS — Homebrew tap
brew tap ronggothelast/nift
brew install nift

# Arch Linux — AUR
yay -S nift-bin

# Debian / Ubuntu — .deb
curl -L https://github.com/ronggothelast/nsm-v2/releases/latest/download/nift_amd64.deb -o nift.deb
sudo dpkg -i nift.deb

# Verify
nift --version
# nift 2.0.0-dev

From source

The build is a CMake preset away. You will need a C++20-capable compiler, CMake 3.25+, Ninja, and the bundled vcpkg submodule. The submodule fetches every other dependency.

Linux / macOS

git clone --recursive https://github.com/ronggothelast/nsm-v2.git
cd nsm-v2
./third_party/vcpkg/bootstrap-vcpkg.sh -disableMetrics

cmake --preset release
cmake --build --preset release
sudo cmake --install build/release

Windows (MSVC)

git clone --recursive https://github.com/ronggothelast/nsm-v2.git
cd nsm-v2
.\third_party\vcpkg\bootstrap-vcpkg.bat -disableMetrics

cmake --preset release
cmake --build --preset release

Docker

The published image is multi-stage Alpine for the build, glibc runtime for the binary. About 18 MB compressed. 

docker pull ghcr.io/ronggothelast/nift:latest

# Build a project mounted from the host
docker run --rm -v "$PWD:/site" -w /site ghcr.io/ronggothelast/nift:latest build

# Dev server, port 8080
docker run --rm -v "$PWD:/site" -w /site -p 8080:8080 \
  ghcr.io/ronggothelast/nift:latest serve --host 0.0.0.0

Verify the install

A working installation prints version, exits 0, and accepts --help:

nift --version
# nift 2.0.0-dev (commit 775cd54)

nift --help
# usage: nift <command> [options]
# commands: build, serve, init, migrate, version

Trouble? Open an issue with the output of nift --version and your CMake configuration log.
