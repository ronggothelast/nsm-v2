Examples

Code examples

Real-world examples for common Nift v2 patterns. Copy, paste, customize.

01 — Minimal site

Single-page site with a template

nift.json
{"name": "minimal", "content_dir": "content"}

templates/default.html
<!DOCTYPE html>
<html>
<head><title>$title</title></head>
<body>
  @content()
</body>
</html>

content/index.html
@template("default")
@set(title, "Home")
<h1>Hello, world!</h1>
02 — Blog with navigation

Multi-page blog with shared header and footer

content/blog/my-post.html
@template("blog-post")
@set(title, "Understanding SIMD")
@set(date, "2026-01-15")

<article>
  <h1>$title</h1>
  <time>$date</time>
  <p>SIMD lets us scan templates in 32-byte batches...</p>
</article>

templates/blog-post.html
@template("default")
@set(layout, "blog")
<div class="blog-layout">
  @include("blog-nav.html")
  @content()
</div>
03 — Conditional content

Show different content based on variables

@set(env, "production")

@if($env == "production") {
  // Production: analytics, minified assets
  <script src="analytics.min.js"></script>
} @else {
  // Development: debug tools
  <script src="debug.js"></script>
}
04 — Loop with @for

Generate repeated HTML structures

@set(items, 5)

<ul>
@for(i; 1; $items + 1) {
  <li>Step $i: Do the thing</li>
}
</ul>
05 — Reusable blocks with @def

Define once, use everywhere

// Define a card component
@def(card, title, desc, link) {
  <article class="card">
    <h3>$title</h3>
    <p>$desc</p>
    <a href="$link">Read more →</a>
  </article>
}

// Use it
@card("SIMD Scanner", "2.4× faster", "/docs/simd")
@card("BLAKE3 Cache", "Incremental builds", "/docs/cache")
06 — Generate data files

Use @write to generate JSON or other data

@set(siteName, "My Blog")
@set(version, "1.0.0")

@write("output/manifest.json") {
  {
    "name": "$siteName",
    "version": "$version",
    "generated": true
  }
}
07 — Lua plugin

Custom build-time logic with Lua

-- plugins/sitemap.lua
function on_build_complete(ctx)
  local pages = ctx.get_pages()
  local xml = "<?xml version='1.0'?>\n<urlset>\n"
  for _, page in ipairs(pages) do
    xml = xml .. "  <url><loc>" .. page.url .. "</loc></url>\n"
  end
  xml = xml .. "</urlset>"
  ctx.write_file("public/sitemap.xml", xml)
end
