Changelog

Nift v2 development milestones. Each phase builds on the previous.

Phase 7 · Packaging & CI

Release automation and cross-platform QA

GitHub Actions CI matrix (Linux, macOS, Windows). Release tarballs with static binaries. AddressSanitizer + UBSan in CI. Exhaustive QA suite: chaos testing (server lifecycle), recovery testing (crash recovery), network testing (HTTP API security via 24 curl-based tests).

272/272 tests ASan+UBSan clean 3-platform CI
Phase 6 · CLI & Plugins

Command-line interface and plugin system

Full CLI with subcommands: init, build, serve, clean, migrate, plugin, bench. Lua plugin system via sol2 with sandboxed execution. C-ABI native plugin loader via dlopen/LoadLibrary. v1 migrator for nsm.config → nift.json.

Phase 5 · Dev Server

Hot-reload development server

Built-in HTTP server via cpp-httplib. Polling file watcher for content changes. Automatic rebuild + livereload on save. Serves from output/ directory with correct MIME types.

Phase 4 · SIMD & mmap

High-performance I/O and scanning

Memory-mapped file I/O via mio. SIMD scanner via xsimd (AVX2): 2.4× faster than scalar for template marker detection. Work-stealing thread pool for parallel builds. Benchmark suite.

Phase 3 · Cache & Lua

Incremental builds and scripting

BLAKE3-keyed build cache (JSON persistence). Lua bridge via sol2 for plugin scripting. Cache invalidation: template changes propagate to all consuming pages.

Phase 2 · Parser

Template language rewrite

Context-aware lexer (only @ and $ special in text mode). Recursive descent parser producing typed AST. Visitor-pattern evaluator. 123 test cases covering all directives, expressions, and edge cases.

Phase 1 · Foundation

Core types and build system

CMake + Ninja + Just build system. vcpkg manifest for dependency management. Core crate: Path, Result/Expected, Filesystem, Str, DateTime types. 62 unit tests. .clang-format, .clang-tidy, .editorconfig.

Phase 0 · Recon

Legacy codebase analysis

Full audit of Nift v1: 65 files, 32,629 LOC. Identified hotpaths (Parser.cpp: 16,949 LOC monolith). Mapped include graph (DAG, no cycles). Documented build system, toolchain requirements, and migration risks. 7 recon documents produced.
