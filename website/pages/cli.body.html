CLI Reference

Complete command-line reference for the nift binary. Run nift --help for a quick overview.

nift init <name>
Scaffold a new Nift v2 project in a directory.
nift init my-site
# Creates my-site/ with nift.json, content/, templates/
--bare Skip sample content (empty project)
--template <url> Use a remote project template
nift build
Build the site. Processes all files in content_dir to output_dir.
nift build
# Built 28 · cached 0 · failed 0 in 42 ms

nift build --verbose
# Shows per-file timing and cache status

nift build --clean
# Deletes output/ and cache before building
--verbose, -v Show per-file details
--clean Clean before build (equivalent to nift clean + build)
--threads, -j <N> Override thread count (default: auto)
--no-cache Force full rebuild, ignore cache
--minify Minify HTML/CSS output
--drafts Include files marked as draft
nift serve
Start the dev server with file watching and livereload.
nift serve
# Serving public/ at http://127.0.0.1:8080
# Watching content/, templates/, partials/ for changes
--port, -p <N> Override port (default: 8080 from config)
--host <addr> Bind address (default: 127.0.0.1)
--no-livereload Disable livereload injection
--open, -o Open browser on start
nift clean
Delete the output directory and build cache.
nift clean
# Removed public/ (28 files)
# Removed .nift/cache.json
nift migrate <path>
Migrate a Nift v1 project to v2 schema.
nift migrate ./old-project
# detected   nsm.config (v1 schema)
# written    nift.json
# preserved  nsm.config.backup
# ok         28 routes, 4 plugins, 0 warnings
--dry-run Show what would be migrated without writing
--force Overwrite existing nift.json
nift plugin <subcommand>
Manage plugins — list, install, create, and inspect.
list List installed plugins with versions
install <path> Install a plugin from a local path or URL
create <name> Scaffold a new Lua plugin with boilerplate
info <name> Show plugin metadata and hooks
nift bench
Run built-in benchmarks (SIMD scanner, build pipeline, cache).
nift bench
# bench_simd_scanner    0.37 ns/byte (1 MiB sparse)
# bench_scalar_scanner  0.89 ns/byte (1 MiB sparse)
# bench_build_cold      95.2 ms (200 pages)
# bench_build_warm      6.1 ms (1 page changed)
Global flags
Flags available on all commands.
--help, -h Show help for the command
--version, -V Print version, commit hash, and build date
--quiet, -q Suppress non-error output
--color <when> Color output: auto, always, never
--config <path> Path to nift.json (default: ./nift.json)
Exit codes
Standard exit codes for scripting and CI.
0 Success
1 Build error (template syntax, missing file, etc.)
2 Config error (invalid nift.json)
3 Plugin error (crash, timeout, sandbox violation)
